상품 목록페이지 라우트 추가하기

상품 목록페이지 라우트를 추가하기 전에 먼저 네비게인션에 라우트를 변경할 수 있게 처리하겠습니다.

리엑트 라우터에서 Link 를 이용하면 브라우저 히스토리를 변경하는 링크를 만들 수 있습니다. 라우팅을 시킬 수 있는거죠. 그리고 url과 비교해서 해당 브라우저의 URL이 to와 매칭이되면 특정 CSS 클래스를 추가할수 있는 NavLink 도 있습니다.

저희는 NavLink를 써서 URL이 해당 링크와 매칭이 되면 active 클래스를 추가하면됩니다. 하지만 a 태그가 아닌 li 태그에 즉 부모 태그에 클래스가 변경되야하기 때문에 NavLink에 그려질 컴포넌트를 직접 전달해야합니다. (참고 https://reactrouter.com/web/api/Link/innerref-function)

이제 NavItem을 다음과 같이 변경하겠습니다.

import React from 'react';
import PropTypes from 'prop-types';
import classNames from 'classnames';
import { NavLink } from 'react-router-dom';

const LinkItem = React.forwardRef((props, ref) => {
  const handleClick = (e) => {
    e.preventDefault();
    e.stopPropagation();
    props.navigate();
  };
  return (
    <li className={classNames('nav-item', props.className)}>
      <a ref={ref} href="" className="nav-link" onClick={handleClick}>
        {props.text}
      </a>
    </li>
  );
});

const NaviItem = ({ to, text }) => {
  return (
    <React.Fragment>
      <NavLink to={to} activeClassName="active" text={text} exact component={LinkItem}></NavLink>
      <style jsx>
        {`
          :global(.nav-item > .nav-link) {
            color: #fff;
            font-family: 'Open Sans', helvetica, arial, sans-serif;
            font-weight: 700;
            letter-spacing: 1px;
            text-transform: uppercase;
          }
          :global(.nav-link) {
            padding-top: 25px;
            border-top: 6px solid transparent;
          }
          :global(.nav-item.active > .nav-link, .nav-item.active > .nav-link:hover) {
            background: transparent;
            color: #fff;
            border-top: 6px solid #1ab394;
          }
          :global(.nav-item > .nav-link:hover) {
            color: #1ab394;
          }
          :global(.active > a:focus) {
            background: transparent;
            color: #fff;
          }
        `}
      </style>
    </React.Fragment>
  );
};
NaviItem.propTypes = {
  to: PropTypes.string.isRequired,
  text: PropTypes.string.isRequired,
  active: PropTypes.bool,
  action: PropTypes.func,
  show: PropTypes.bool,
};
export default NaviItem;

레이아웃 컴포넌트 만들기

ProductList/index.js 만들어서 하다보면 큰 어려움에 부딪히게 됩니다. 바로 카트에 대한 상태정보를 어떻게 사용하는건가 입니다. Home 페이지 컴포넌트처럼 모든 내용을 복사하고 별도의 상태를 가져가면 문제가 카트 상태정보가 공유가 안됩니다. 그러면 카트 정보를 App.js로 올리는걸 생각해볼 수 있습니다. (리덕스가 정말 팔요하죠!)

하지만 이걸 해결하기도하고 네비게이션과 카트등은 각 페이지마다 모두 공통으로 보여지는 영역입니다. 단지 <main> 안에 특정 부분이 변경되기 때문에 어떻게보면 같은 레이아웃을 공유하고 그 레이아웃에서 특정 부분만 바뀌는걸 고려할 수 있습니다.

그러면 이제 Layout 컴포넌트를 만들어봅시다! 다음과 같이 lyaouts 폴더를 만들고 PublicLayout이라고 폴더를 만들고 그안에 index.js를 아래와 같이 만들어 보겠습니다.

/* eslint-disable react/prop-types */
import React from 'react';
import { Route } from 'react-router-dom';
import Drawer from '@/components/Drawer';
import Cart from '@/pages/Home/Cart';
import Footer from '@/pages/Home/Footer';
import Navitation from '@/components/Navigation';

export default class PublicLayout extends React.Component {
  state = {
    isShopingCartOpen: false,
    cartItems: [],
  };
  handleCartClicked = () =>
    this.setState({
      isShopingCartOpen: !this.state.isShopingCartOpen,
    });

  handleCheckout = ({ items, total }) => {
    console.log(`Checkout ${JSON.stringify(items)} total: ${total}`);
  };

  handleCartClosed = () => {
    this.setState({ isShopingCartOpen: false });
  };

	handleCartItemRemoved = ({ id }) => {
    const foudnItem = this.state.cartItems.find((v) => v.product.id === id);
    if (foudnItem == null) {
      throw new Error(`Can not find the item (${id})`);
    }
    if (foudnItem && foudnItem.counts > 1) {
      foudnItem.counts -= 1;
    } else {
      const index = this.state.cartItems.indexOf(foudnItem);
      this.state.cartItems.splice(index, 1);
    }
    this.setState({
      cartItems: [...this.state.cartItems],
    });
  };

  render() {
    const { component: Component, ...rest } = this.props;

    return (
      <Route
        {...rest}
        render={(matchProps) => (
          <React.Fragment>
            <Drawer
              isOpen={this.state.isShopingCartOpen}
              component={Cart}
              items={this.state.cartItems}
              onItemRemove={this.handleCartItemRemoved}
              onCheckout={this.handleCheckout}
              onClose={this.handleCartClosed}
            />
            <Navitation onCartClick={this.handleCartClicked} cartItemCounts={this.state.cartItems.length} />
            <Component {...matchProps} />
            <Footer />
          </React.Fragment>
        )}></Route>
    );
  }
}

    isShopingCartOpen: false,

  };

  // 생략
